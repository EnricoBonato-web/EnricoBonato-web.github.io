VS Code Extensions and Installation Commands

  • christian-kohler.path-intellisense: Autocomplete filenames in your project.
    Install Command: code --install-extension christian-kohler.path-intellisense
  • eamodio.gitlens: Supercharge the Git capabilities built into Visual Studio Code.
    Install Command: code --install-extension eamodio.gitlens
  • esbenp.prettier-vscode: Format code with Prettier in Visual Studio Code.
    Install Command: code --install-extension esbenp.prettier-vscode
  • figma.figma-vscode-extension: Design and prototyping tool integration with Figma and Visual Studio Code.
    Install Command: code --install-extension figma.figma-vscode-extension
  • formulahendry.auto-rename-tag: Auto rename paired HTML/XML tag in Visual Studio Code.
    Install Command: code --install-extension formulahendry.auto-rename-tag
  • GitHub.vscode-pull-request-github: Pull request and issue browsing for GitHub.
    Install Command: code --install-extension GitHub.vscode-pull-request-github
  • glenn2223.live-sass: Compile Sass or Scss to CSS in real-time.
    Install Command: code --install-extension glenn2223.live-sass
  • Gruntfuggly.todo-tree: Show TODO, FIXME, etc. comments in a tree view.
    Install Command: code --install-extension Gruntfuggly.todo-tree
  • kisstkondoros.vscode-gutter-preview: Preview images and color values in the gutter.
    Install Command: code --install-extension kisstkondoros.vscode-gutter-preview
  • michelemelluso.code-beautifier: Beautify your source code in Visual Studio Code.
    Install Command: code --install-extension michelemelluso.code-beautifier
